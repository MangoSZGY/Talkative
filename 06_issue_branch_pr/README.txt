
06. PROJEKT – Issue, Branch, Pull Request munkafolyamat
-------------------------------------------------------
Feladatok:
1. Hozzatok létre egy issue-t: "Kereső funkció hozzáadása".
2. Címkék: feature, backend
3. Delegáljátok csapattársnak.
4. A megbízott branch-et hoz létre: feature/search
5. Lokális fejlesztés, több commit.
6. Pull request nyitása → reviewer kéri módosításokat.
7. Módosítások elvégzése, új commit.
8. Reviewer approve → merge → branch törlése.
9. Kanban board frissítése (Backlog → In Progress → Review → Done).
